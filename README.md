# Site Manager  

## 前台
訪問前台及使用功能時，歡迎訪問者註冊屬於自己的帳密，  
也可以考慮使用測試帳號登入，帳密如下：  
帳號：123@456.com  
密碼：123456  
  
**附註：**  
- 雖然後台沒有以明碼儲存密碼，但註冊時仍不建議使用您平時使用的帳密
- 每次重啟Render伺服器時，資料將會重製，故您註冊的帳號一段時間後就消失屬於正常現象

## 後台
欲訪問後台及使用管理者功能時，可使用測試帳密登入，帳密如下：  
帳號：111@111.com  
密碼：123123  

## 說明
**附註：**  
此專案使用了Render來模擬部署後端與資料庫，首次開啟網頁時可能會需要等30秒左右才能使伺服器完成暖機，若有不便敬請見諒。
